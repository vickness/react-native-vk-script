# react-native-vk-script

## 安装

`yarn add react-native-vk-bugly`

## 使用


```bash
// 设置icon和launch图片
x-image <icon/launch> path
```

```bash
// 打包app，需要先配置好证书
x-pack <staging/release> <ios/android>
```

```bash
// 发布热更新(CodePush)
x-update <staging/release> <ios/android>
```

```bash
// app重签名
x-sign
```
